# 開始喚醒

自動啟動模擬器，自動啟動用戶端，自動進入遊戲。

自動啟動模擬器需額外設定，詳見啟動設定，且需勾選連線設定內的 `ADB 連線失敗時嘗試啟動模擬器`。

## 帳號切換

本功能需與切換配置或定時執行一起使用，會在每次執行 `開始喚醒` 任務時都進行一次帳號切換操作。

僅支援切換至已登入的帳號，使用登入名稱進行查找，請保證輸入內容在所有已登入帳號中是唯一的。

- 範例：
  - 官服：`123****8901`，可輸入 `123****8901`、`123`、`8901`、`3****8`
  - B 服：`張三`，可輸入 `張三`、`張`、`三`
