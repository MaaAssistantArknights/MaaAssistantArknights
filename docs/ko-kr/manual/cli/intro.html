소개

MaaAssistantArknights2025년 7월 15일1분 미만

소개

Rust로 작성된 간단한 MAA 명령줄 도구입니다.

기능

  • maa fight, maa run <task>와 같은 미리 정의된 또는 사용자 정의 작업을 실행합니다.
  • maa install 및 maa update를 사용하여 MaaCore 및 리소스를 설치하고 업데이트합니다.
  • maa self update를 사용하여 자체 업데이트를 수행합니다.
