MAA




MAA 的意思是 MAA Assistant Arknights

一款明日方舟遊戲小助手

基於圖像辨識技術，一鍵完成全部日常任務！

絕讚更新中 ✿✿ヽ(°▽°)ノ✿

下載與安裝

請閱讀 文件 後前往 官網 或 Releases 下載，並參閱 新手上路 進行安裝。

亮點功能

  • 刷理智，掉落辨識及上傳 企鵝物流，一圖流
  • 智慧基建換班，自動計算幹員效率，單設施內最佳解；同時也支援 自定義排班
  • 自動公招，可選使用加急許可，一次全部刷完！公招數據自動上傳 企鵝物流，一圖流
  • 支援手動辨識公招介面，方便對高星公招做出選擇 （你的這個高資回費出的是推進之王呢還是推進之王呢）
  • 支援辨識幹員列表，統計已有和未有幹員及潛能，並在公招辨識顯示
  • 支援辨識養成材料，並匯出至 企鵝物流刷圖規劃、明日方舟工具箱、ARK-NIGHTS 幹員培養表
  • 造訪好友、獲取信用及購物、領取日常獎勵等，一鍵全日常自動長草
  • 肉鴿全自動刷源石錠和等級，自動燒水和凹直升，智慧辨識幹員及練度
  • 選擇作業 JSON 檔案，自動抄作業， 影片展示
  • 支援 C, Python, Java, Rust, Golang, Java HTTP, Rust HTTP 等多種介面，方便整合呼叫，自定義你的 MAA！

話不多說，看圖！

使用說明

功能介紹

請參閱 用戶手冊。

外服支援

目前國際服（美服）、日服、韓服、繁中服的絕大部分功能均已支援。
但由於外服用戶較少及項目人手不足，很多功能並沒有進行全面的測試，所以請自行體驗。
若您遇到了 Bug，或對某個功能有強烈需求，歡迎在 Issues 和 討論區 催更；或加入我們一起建設 MAA！請參閱 外服適配教學。

CLI 支援

MAA 支援命令列介面（CLI）操作，支援 Linux，macOS 和 Windows，可用於自動化指令碼或在無圖形介面的伺服器上使用。
請參閱 CLI 使用指南。

DirectML 支援說明

本軟體支援 GPU 加速功能，其在 Windows 平台上依賴於 Microsoft 提供的獨立元件 DirectML。
DirectML 並非本專案的開源部分，也不受 AGPL 3.0 的約束。
為方便用戶使用，我們隨安裝包附帶了一個未經修改的 DirectML.dll 檔案。
如果您無需 GPU 加速功能，可安全刪除該 DLL 檔案，軟體的核心功能仍可正常執行。
